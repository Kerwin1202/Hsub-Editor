영상-자막 연결

enter
선택된 자막을 적용한다. 종료시간이 없을 경우 현재시간을 종료시간으로 설정
shift + enter
줄바꿈

tab
다음 자막으로 이동. 다음 자막이 없을 경우 현재 시간을 시작시간으로 새 자막 생성
shift + tab
이전 자막으로 이동

alt + insert
현재 시간을 시작시간으로 새 자막 생성
alt + end
선택된 자막의 종료시간을 시작시간으로 새 자막 생성
alt + pageup
선택된 자막의 시작시간을 현재시간으로 설정
alt + pagedown
선택된 자막의 종료시간을 현재시간으로 설정

ctrl + space
일시정지
ctrl + ←
1초 뒤로
ctrl + →
1초 앞으로
